# 渔港船舶渔获与配额管控平台

> 我国近海小型渔船数量庞大，传统纸质渔捞日志与电子秤台账难以满足渔业资源总量控制与可追溯管理要求。本平台提供出海申报 → 船位追踪 → 渔获登记 → 配额管控 → 执法办案的全链路能力，支持 JT/T 808 主流船位终端协议，弱网移动端友好，容器编排一键启动。

## 业务角色

- **船东船长**：移动端出海申报、归港扫码提交渔获预申报
- **渔港管理员**：管理端审核申报、确认过磅、处理告警
- **海洋与渔业局配额管理人员**：维护年度配额、查看按海区/船东/渔港的累计与剩余
- **海警渔政执法办案人员**：移动端查询船舶历史、开具电子违规告知书
- **水产采购商**：回传采购品种/重量/价格/目的地

## 核心能力

| 模块 | 能力描述 |
|------|----------|
| 出海前 | 校验证件有效期、违规停业整改、前次未归港超时；记录作业人员、计划海域、网具规格 |
| 出海期间 | 持续回传船位 → 轨迹绘制；越界、长时间失联、关闭终端等异常自动告警 |
| 归港后 | 移动端扫码提交渔获预申报单；电子秤台账确认过磅；偏差>10% 触发原因填报与人工复核 |
| 采购 | 采购商回传品种/重量/价格/目的地加工企业 |
| 配额管控 | 自动核算每艘船、每位船东、每个作业海域、本渔港、本海区的累计渔获与剩余配额；配额耗尽时禁止下一航次申报 |
| 执法办案 | 移动端查询历史申报/轨迹/渔获；开具电子违规告知书，扣减对应配额 |

## 技术栈

- 后端：Spring Boot 3.2 + Java 17 + JPA + PostgreSQL（开发期 H2 内存库）
- 缓存：Redis 7
- 管理端：Vue 3 + Vite + Element Plus + ECharts
- 移动端：Vue 3 + Vite + Vant（适配渔港码头弱网）
- 反向代理：Nginx
- 模拟器：Python 3.11（100 艘船 × 30 天）
- 编排：Docker Compose

## 启动方式

### 方式一：Docker 一键启动（推荐）

```bash
bash start-and-check.sh
```

启动后访问：
- 管理端：http://localhost:8081
- 移动端：http://localhost:8082
- API 根：http://localhost:8080/api

模拟器将自动注入 100 艘船 × 30 天的连续出海与渔获数据。

### 方式二：本地开发模式

#### 后端
```bash
cd backend
mvn spring-boot:run
```
默认使用 H2 内存库，无需额外配置即可启动；如需 PG，请设置环境变量 `SPRING_PROFILES_ACTIVE=docker` 并确保 `postgres:5432` 可用。

#### 管理端
```bash
cd admin-web
npm install
npm run dev
```
访问 http://localhost:5173

#### 移动端
```bash
cd mobile-web
npm install
npm run dev
```
访问 http://localhost:5174

#### 模拟器
```bash
cd simulator
pip install -r requirements.txt
python simulator.py --vessels 100 --days 30
```

## 典型场景自检

启动完成后可在管理端执行下列自检：

1. **100 艘船 × 30 天数据** — 模拟器执行完成后，访问【船舶档案】可见 100+ 艘船
2. **配额查询** — 进入【配额管控】，输入任一船东姓名，可查询到该船东年度累计渔获与剩余配额
3. **告警链路** — 进入【告警事件】，可见【超出申报海域】【关闭船位终端】等自动产生的告警
4. **执法办案** — 进入【执法办案】，可见模拟器开具的电子违规告知书，已关联航次与配额扣减
5. **追溯链路** — 任一航次申报单、渔获单、违规告知书、告警均可在管理端与移动端通过船号/航次号关联

## 海域覆盖与对接路径

- **第一阶段**：覆盖本省沿海岸段主要渔港与作业海区（已实现 5 个示范渔港）
- **第二阶段**：对接海洋与渔业局年度可捕捞总量与禁渔公告数据源（`/api/quota/rule/save` 接口已就绪）
- **第三阶段**：对接海警渔政执法办案系统与水产采购商加工销售监管数据源（`/api/enforcement/*`、`/api/purchase/*` 接口已就绪）

## 字段命名约定

所有船号、海区编号、字段说明均使用自然语言，例如：
- 船号：`泉州-渔-2001`、`福州-渔-2010`
- 渔港：`泉州祥芝渔港`、`福州连江黄岐渔港`
- 海区：`闽南近海渔区`、`闽东近海渔区`、`闽中近海渔区`
- 申报单号：`渔港名-船号-YYYYMMDD-序号`
- 告知书编号：`WZ-船号-YYYYMMDD-序号`

## 配额与告警阈值

- 偏差告警阈值：10%（可调整 `CatchService.DEVIATION_THRESHOLD`）
- 最长允许出海天数：30 天（可调整 `VoyageService.DEFAULT_MAX_DAYS`）
- 失联判定：> 120 分钟未回传船位（可调整 `AlertService`）
